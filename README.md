# Cuda_Study
This repository is for studying CUDA.

How to build?

    > mkdir build && cd build
    > cmake .. && cmake --build .
    > ./CUDA_STUDY
    
The result is output like this:

    SAXPY execution time : 4.12131ms  # This is CUDA Result
    SAXPY execution time : 0.002048ms # This is Muti Threading Result
